# Adult Census Income Classification

## Project Overview

This project predicts whether an individual's annual income exceeds $50,000 using the Adult Census Income dataset.

## Dataset

- Source: UCI Machine Learning Repository / Kaggle
- Dataset: Adult Census Income

## Machine Learning Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Libraries Used

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Workflow

1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Data Preprocessing
5. Model Training
6. Model Evaluation
7. Conclusion

## Best Model

The model with the highest accuracy was selected as the best-performing classifier.
